# phonedirectory
Gestion de Répertoire téléphonique
Ce projet est une application d'annuaire téléphonique implémentée en langage C. Elle permet aux utilisateurs de gérer un répertoire téléphonique en ajoutant, recherchant, modifiant et supprimant des informations de contact.

## Fonctionnalités
- Ajouter de nouveaux contacts au début, à la fin ou à une position spécifique du répertoire.
- Rechercher des contacts par nom ou numéro de téléphone.
- Mettre à jour ou modifier les informations des contacts existants.
- Supprimer des contacts du répertoire.
- Sauvegarder et charger le répertoire à partir d'un fichier.
- 
## Prérequis
- GCC ou un autre compilateur C.
- Un terminal ou un IDE pour exécuter le code.

## Comment exécuter le projet
1. Clonez ce dépôt :
git clone https://github.com/ronin-tn/phonedirectory.git
2. Accédez au dossier du projet :
cd phonedirectory
3. Compilez le code :
gcc -o annuaire_telephonique main.c *.c
4. Exécutez le programme :
./annuaire_telephonique
